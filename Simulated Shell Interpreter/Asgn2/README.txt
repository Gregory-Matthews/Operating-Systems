Description:
-------------------------------------------------------------------------------------------
The assignment is in regards to Project 1: Programming Projects 
from the book "Operating Systems Projects 9th Edition" by Abraham Silberchatz. (pg. 157-159)
The source code (shell.c) includes additional functionality by adding:
 1.) Alias feature, for example: alias dir "ls -al"
 2.) Set Path feature, for example: set path = (. /bin /usr/bin)
 3.) Ability to intiliaze the shell interpreter with an initilization file on startup (.cs543rc) 
 4.) Ability to run on diagnostics in the shell, for example: set verbose on

How to Run:
--------------------------------------------------------------------------------------------
 1.) In the terminal, go into folder: /gmmathews_hw1
 2.) Type into the terminal: gcc shell.c -o shell
 3.) An executable will be created named: shell
 4.) Execute the program by typing: ./shell
 5.) A shell interpreter will wait for user input as shown by: osh>
